# Geo-coordinate-converter
This is a simple code to convert between different coordinate systems 

The code has been written in CLI based format 
The user need to have a python3 or basde python format to run it 



The user can save the Main.py into his system and run :
# Python3 Main.py 
The code will run in the backgound and a cli based interface will pop for the user to choose the desired conversion type.
